# BankLoanApproval
AI project for approving loans

Dataset from Kaggle: https://www.kaggle.com/datasets/vikramamin/bank-loan-approval-lr-dt-rf-and-auc/data

Use python 3.7 or later. 
Uses the following Libraries
- pandas
- numpy
- matplotlib
- seaborn
- tensorflow
- keras
- scikit-learn
- joblib
- streamlit
